# Speechify-Style QA Practice Repository

This is a **practice repository** that simulates the kind of take-home / first-round technical assessment described in the email you received.

It is designed to help you practice:
- opening a repository on GitHub
- cloning it locally
- editing markdown files
- answering QA fundamentals questions
- writing API test cases
- completing a small optional coding task
- organizing your answers like a real candidate submission

## Repository structure

- `INSTRUCTIONS.md` – assessment rules and submission guidance
- `tasks/` – assignment tasks
- `resources/` – API context and sample data
- `answers/` – blank answer templates you can fill in
- `reference_solution/` – example answers for learning and self-checking

## Suggested practice flow

1. Read `INSTRUCTIONS.md` first.
2. Open each file in `tasks/`.
3. Fill in the corresponding file in `answers/`.
4. Commit your changes.
5. Push the repository to your GitHub.
6. Compare your work against `reference_solution/` only after you finish.

## Important

This repo is for **practice only**.
For a real assessment, follow the employer's rules exactly.
